Philosopher i

1: void philosopher(int i) {
2: while(true) {
3: think();
4: wait(chopstick[i]);
5: wait(chopstick[(i+1)%5]);
6: eat();
7: signal(chopstick[i]);
8: signal(chopstick[(i+1)%5]);
9: }
10:}

🧠 Deadlock Condition

Deadlock occurs if ALL philosophers pick up their LEFT chopstick simultaneously — none can pick up the right one.

🛡️ Prevention

Allow at most 4 philosophers to sit simultaneously, or use an asymmetric solution (odd picks left first, even picks right first).

Reader

1: void reader() {
2: wait(mutex);
3: readcount++;
4: if(readcount == 1) wait(wrt);
5: signal(mutex);
6: read_data();
7: wait(mutex);
8: readcount--;
9: if(readcount == 0) signal(wrt);
10: signal(mutex);
11:}

Writer

1: void writer() {
2: wait(wrt);
3: write_data();
4: signal(wrt);
5:}

🧠 Starvation Risk

In Reader-Priority: if readers keep arriving, writers may starve indefinitely.

🛡️ signal(wrt) wake-up

When the last reader exits (readcount==0), it calls signal(wrt) to WAKE UP any blocked writer.
